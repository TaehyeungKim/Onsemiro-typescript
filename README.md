## Stack
✓ Core: React.js + Typescript + Django <br/>
🎨 Styling: TailwindCSS <br/>
✍️ State Management: Recoil <br/>
🚀 Deploy: Vercel, AWS <br/>

## Feature

회원가입 - 사용자 추천 시 사용자의 선호/성향을 반영하기 위해 설문 형태로 사용자의 정보 수집 <br/>
